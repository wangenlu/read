# 概述

核心：需要知道分层和分用的目的，让各层只关注自己的事情。

- 分层
- TCP/IP分层
- 封装
- 分用 Demultiplexing
- 端口号
- RFC

连接网络的另一个途径是使用网桥。网桥是在`链路层`上对网络进行互连，而路由器则是
在`网络层`上对网络进行互连。网桥使得多个局域网（LAN）组合在一起，这样对上层来说就
好像是一个局域网。
