# Dubbo

- [dubbo](https://github.com/apache/dubbo)

## Dubbo version

`Dubbo` 版本: **2.7.2** 目前 `Dubbo` 还在 `apache` 基金孵化 不同的版本之间的改动还是很大的

```xml
<dependency>
    <groupId>org.apache.dubbo</groupId>
    <artifactId>dubbo-bom</artifactId>
    <version>2.7.2</version>
    <type>pom</type>
    <scope>import</scope>
</dependency>
```
